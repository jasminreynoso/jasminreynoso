# Pixel Art Generator

Create pixel art with a customizable grid. Click pixels to color them, or use the fill option to color the entire canvas.

## Features
- Customizable grid size (4x4 to 64x64)
- Color picker
- Click to color individual pixels
- Fill all pixels option
- Clear canvas
- Save as PNG image

## Technologies
- HTML5 Canvas
- JavaScript
- CSS3 Grid
